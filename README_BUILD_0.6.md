# ISO 19011 Academy · Build 0.6

## Professional Visual Edition

Este build incrementa de forma importante la presencia visual del curso sin modificar la autenticación, Supabase, progreso ni evaluaciones existentes.

### Cambios principales

- 5 portadas visuales, una para cada Parte.
- 10 ilustraciones temáticas insertadas dentro de las lecciones.
- Ilustración específica para la auditoría simulada.
- Recursos SVG locales: no dependen de internet y conservan nitidez en cualquier pantalla.
- Mejor jerarquía tipográfica y separación visual del contenido.
- Diseño responsive para escritorio y móvil.

### Publicación

Reemplazar los archivos del repositorio y ejecutar:

```powershell
git add .
git commit -m "Build 0.6 Professional Visual Edition"
git push
```

Después de publicar, hacer una recarga forzada con `Ctrl + F5`.
