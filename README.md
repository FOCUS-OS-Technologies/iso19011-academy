# Curso HTML ISO 19011:2018

## Contenido
- 5 partes de aproximadamente 2 horas cada una.
- Teoría resumida y estructurada.
- Autoevaluaciones con retroalimentación inmediata.
- Seguimiento de progreso mediante `localStorage`.
- Constancia imprimible.
- Diseño adaptable para PC, proyector y móvil.

## Cómo abrir
1. Descomprime la carpeta.
2. Abre `index.html` con Chrome o Edge.

## Cómo gestionar en Visual Studio Code
1. Abre Visual Studio Code.
2. Selecciona **Archivo > Abrir carpeta**.
3. Abre la carpeta `ISO19011_HTML_Curso_Completo`.
4. Instala opcionalmente la extensión **Live Server**.
5. Haz clic derecho en `index.html` y selecciona **Open with Live Server**.

## Archivos principales
- `index.html`: estructura de la aplicación.
- `css/style.css`: identidad visual.
- `js/course-data.js`: contenido de las 5 partes y preguntas.
- `js/app.js`: navegación, progreso, evaluaciones y constancia.

## Edición de contenido
En `js/course-data.js`, cada parte contiene:
- `title`
- `duration`
- `objectives`
- `lessons`
- `quiz`

## Consideraciones
- El curso es una síntesis formativa; no reproduce el texto completo de ISO 19011:2018.
- La norma oficial debe adquirirse y consultarse por separado.
- El progreso se guarda en el navegador actual. Si se borra el almacenamiento del navegador, se reinicia.
